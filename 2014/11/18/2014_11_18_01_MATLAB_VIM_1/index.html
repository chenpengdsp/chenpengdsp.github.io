前言

如果你觉得MATLAB自带的编辑器对你而言足够了，那么请忽略本文，如果你想更高效的书写MATLAB程序，而且有足够的学习时间，可能要稀稀疏疏的把玩一周，你的代码书写会有质的飞跃。

注意：本文所讨论的是Windows操作系统。

VIM 基本教程

VIM 安装

VIM 安装，请移步VIM官方网站（ http://www.vim.org/download.php#pc ）下载，本文所使用的是Windows下的gvim，即官网中的“PC: MS-DOS and MS-Windows”分类，下载安装文件gvim74.exe.

安装注意事项：

  1. 建议安装在盘根目录（如D:Vim)；
  2. 选择完全版本安装。

VIM 基本操作

VIM编辑器的学习曲线是陡峭的，如果想在几天内成为高手是基本不可能的，所以我们需要一边使用一边学习，下面我介绍一些常用的VIM操作命令。

  • Normal与Insert模式

    1. VIM编辑器将文本的编辑书写过程分为两个模式，即Normal和Insert模式，一般情况下，我们在Insert模式输入内容，在Normal模式下编辑内容。

    2. 按下i键，从Normal模式（打开VIM时默认模式）转换为Insert模式，便可像普通记事本一样进行输入操作。按下Esc键，便可返回Normal模式。本文所述编辑命令都是在Normal模式下进行的。

  • 插入命令（i，I，a，A）

    1. 我们已经知道了从Normal模式转换为Insert模式是i键，其实i键的意思是在光标所在位置之前插入文本，如果我们想在光标所在位置之后插入文本，则需要按下a键。

    2. 如果光标在一行的中间，而我们想在本行的开头处插入本文，则要按下I键（即Shift+i）。相应地，如果我们想在本行的末尾处插入文本，则要按下A键（即Shift+a）。

  • 移动命令（h, j, k, l）

    1. VIM 比其他编辑器高效的主要原因在于，我们可以在整个代码书写的过程中，不离开键盘，不用鼠标，而其中最主要的便是高效移动命令。

    2. 在Normal模式下（任何时候按下Esc键，进入Normal模式），按下h键向左移动光标，按下l键向右移动光标，按下k键，光标移动到上一行，按下j键，光标移动到下一行。

    3. 在Normal模式下，按下gg键，移动到文件最开始位置，按下G（Shift+g）移动到文件的最末尾位置。

    4. 在Normal模式下，按下10G（Shift+g）键，移动到文件第10行。

  • 翻页命令

    1. 在Normal模式下，Ctrl+f 表示向前翻一页，Ctrl+b 表示向后翻一页。

  • 定位命令（f, F）

    1. ^ (Shift+6) 光标移动到一行的文字最开始位置。

    2. $ (Shift+4) 光标移动到一行的最末尾位置。

    3. 0 光标移动到一行的最开始位置。

    4. 在一行之中向前定位到指定位置，首先按下f，然后按下需要定位位置的字母，例如以下句子，光标位于i的位置，我们如果想定位到t位置，则需按下ft即可。

      There is a test.

    5. 而F（Shift+f）是先前搜索定位，还是之前的例子，此时光标在t的位置，如果我们想定位到i位置，需要输入Fi即可。

  • 选择命令

    1. 在Normal模式下，按下v进入选择模式，此时再按下移动命令，如h、j、k、l便可选择文本。

    2. 在Normal模式下，按下V (Shift+v) 进入按列选择模式，此时再按下移动命令，如j、k便可按列选择文本。

    3. 在Normal模式下，按下 Ctrl+q 进入按列选择模式，此时再按下移动命令，如h、j、k、l便可选择矩形文本内容。

    4. 由于VIM的思想是命令的组合，例如有如下句子，光标在i位置我们想选择到t位置，便可按下v键，进入选择状态，在按下f键，进入查找状态，在按下响应的查找字母t，便可选择文本“is a t”。

      There is a test.

  • 删除命令

    1. 在Normal模式下，删除一个字母命令x。

    2. 在Normal模式下，删除一整行命令dd。

    3. 在Normal模式下，按下v（或者V，Ctrl+q）进入选择模式，并通过移动命令选择一段文本，此时按下d键，变可删除此段文本。

    4. 如果要改写一整行的内容，按下cc，便删除一整行，并进入插入模式。

  • 复制命令

    1. 在Normal模式下，复制命令y。此命令可以结合选择命令，即通过按键v进入选择命令，并选择部分文本，再按下y，便可复制选择的文本。

    2. 在Normal模式下，复制一整行命令yy。

  • 更改命令

    1. r 将光标所在位置的字母进行更改。如This is a tbst, 光标在b位置，我们按下re，便可将b改为e。

    2. R (shift+r) 如果有连续的多个字母需要更改，如This is a tght, 光标在b位置，我们按下 R (shift+r) ，进入更改模式，再依次输入es，最后按下Esc，便可将tght改写为test。

  • 粘贴命令

    1. p 命令为粘贴命令，在VIM中进行复制（y）和删除（d，x）操作后的内容会保存到内存中，此时按下p键，便可将内容中的文件内容粘贴到光标之后位置。

    2. P (Shift+p) 命令为粘贴命令, 可将内容中的文件内容粘贴到光标之前位置。

  • 缩进命令

    1. >> 命令使本行代码向右缩进。

    2. << 命令使本行代码向左缩进。

    3. 缩进一段代码，按下V进入行选择状态，通过j、k操作选择一段代码，再按下 > 向右缩进，按下 < 向左缩进。

  • 标记书签命令

    1. m命令为书签命令，如ma在光标位置设置名为a的书签，同理mb设置名为b的书签。

    2. `a 命令移动光标到书签a处，`b命令移动光标到书签b处。

    3. ` . 命令移动光标到刚刚修改的内容位置。

  • 代码折叠

    1. za 命令折叠与展开一段代码。

    2. zM 命令折叠所有代码。

    3. zR 命令展开所有折叠。

  • 代码补全

    1. 在插入模式下，输入变量的前几个单词之后，按下Ctrl+n，出现代码补全窗口，继续按Ctrl+n，进行向下选择补全，按下Ctrl+p 向上选择补全。

    2. 在插入模式下，输入一行代码的前几个单词之后，按下Ctrl+x, Ctrl+l, 出现整句代码补全窗口，继续按Ctrl+n，进行向下选择补全，按下Ctrl+p 向上选择补全。

  • 其他命令

    1. ~ 改变字母大小写。

    2. u 命令撤销更改，返回比较旧的结果。

    3. Ctrl+r 取消刚刚u命令的更改，得到比较新的结果。

    4. 在Normal模式下，按下:e test.txt，再按回车键，便可新建一个test.txt文档。

    5. 在Normal模式下，按下:w，再按回车键，便可保存文件。

  • 高级命令组合

    1. 注释一段代码：在Normal模式下，光标定位到第一行代码前面，按下Ctrl+q进入列选择模式，再按下j，向下选择一列文本，在按下I（Shift+i），进入插入模式，输入% ，再按下Esc，便可在一段代码前面都加上% ，注释该段代码。

    2. 删除一个单词，并写入其他替换单词：在Normal模式下，光标定位到该单词，按下ciw即可，其中c表示修改，i表示内部，w表示单词。

    3. 复制括号中内容：在Normal模式下，光标定位到括号中，如果是小括号，按下yi), 如果是中括号，按下yi], 如果是大括号, 按下yi}, 其中y表示复制，i表示括号内，)、]、}表示相应的括号。
