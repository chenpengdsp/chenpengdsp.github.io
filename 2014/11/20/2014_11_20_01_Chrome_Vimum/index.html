Chrome 插件 Vimum 的介绍与安装

各位 Vimer，现在存在一个 Chrome 插件 Vimum，可以将你的 Chrome 变成 VIM 一样的快捷操作。首先安装 Vimum，下载地址：https://chrome.google.com/webstore/detail/vimium/dbepggeogbaibhgnhhndojpepiihcmeb ，安装完之后重启 Chrome。

Chrome 插件 Vimum 的使用

浏览网页的时候，键入?，便可得到帮助菜单，如下图：

主要的快捷键操作有：

  • <Esc>： 如果快捷键不起作用，请<Esc>

  • f： 在当前页面打开连接

  • Shift + f：在新页面打开连接

  • /：搜索，不过只支持英文，n和N分别表示向下和向上搜索

  • o：在当前页面打开连接（智能补全历史记录以及标签等）

  • b：在当前页面打开书签（需要输入书签名）

  • Shift+b：在新页面打开书签（需要输入书签名）

  • Shift+h：回退上一个历史页面（相当于浏览器中的向左箭头）

  • Shift+l：回到下一个历史页面（相当于浏览器的向右箭头）

  • Shift+k，gt：跳转到右边的一个标签页

  • Shift+j，g+Shift+t：跳转到左边的一个标签页

  • t：创建一个新的标签页

  • x：关闭当前的标签页

  • Shift+x：恢复刚刚关闭的标签页

  • d：下翻页

  • u：上翻页

  • gg：移至页顶

  • Shift+g：移至页底

  • r：刷新页面

  • j、k、h、l：向下/上/左/右滚动

  • zi 和 zo：放大/缩小
